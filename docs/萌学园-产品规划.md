# 萌学园 · 产品演进规划

> 从「古诗启蒙小程序」成长为「儿童启蒙 + 亲子成长社区」
> 一句话定位：**陪 3-8 岁孩子启蒙的地方，也是家长记录、分享、抱团的社区。**

---

## 一、定位与人群

- **产品定位**：以古诗启蒙为起点，做"学习 + 家长分享"的亲子成长社区。对标参考：亲宝宝（家庭/社区） + 婷婷诗教/常青藤爸爸（内容/启蒙）。
- **核心年龄**：孩子 **3-8 岁**（学龄前 + 小学低年级）。"萌"定调偏低龄、可爱，上限约 8-9 岁。
- **真实用户**：**家长**（掏钱、使用、传播的主体）。启蒙阶段家长最焦虑、最舍得投入、最爱分享——这是黄金人群。

## 二、为什么是"学习 + 社区"

- 纯工具：孩子学完即走，留存差、难传播。
- 加社区：家长"**为了看别人、为了晒自己**"反复打开 → 从"用一下"变"离不开"。
- 社区是别人**抄不走**的护城河（内容能抄，家长之间的关系网抄不走）。

## 三、分阶段路线图

### 阶段 0 · 现状（已具备的底子）
- **学习闭环**：听官方朗读 → 跟读 → 背诵闯关 → AI 评分 → 复习 → 今日计划
- **内容**：164 首古诗（配图 + 真人音频 + 诗人故事）
- **玩法**：磨耳朵听单、诗词游戏（飞花令/配对）、古诗卡册、在线涂色
- **社区雏形**：图文动态（最多 6 图）+ 点赞 + 分享 + 审核 + 我的动态/编辑
- **留存**：连续打卡、睡前提醒、家长周报
- **增长**：邀请裂变（带邀请码分享）
- **数据**：核心事件埋点 + 管理员数据看板

### 阶段 1 · 验证 0→1：把"古诗启蒙 + 家长社区"跑透
- **目标**：跑出**第一批活跃家长**（几十到几百），验证社区能不能转起来。
- **要做**：
  - 社区冷启动：种子内容运营（自己先发、邀请几个真实宝妈）、给前几条动态点赞评论造氛围
  - 打磨核心学习体验（别再盲目加新功能，用数据看哪些真有人用）
  - 发到宝妈群/朋友圈拉种子用户
- **关键指标**：周活家长数、社区发帖率、次周留存、邀请转化率
- **过关标准**：有一小撮家长**每周主动来 + 主动发动态**（说明社区有了第一口气）

### 阶段 2 · 留存 + 裂变 1→10：让社区转起来
- **目标**：社区自循环，家长口碑带新。
- **要做**：
  - 社区机制：话题标签、活动/比赛（背诗大赛、诗配画征集、打卡挑战）
  - 互动升级：评论、关注、@、家长互助问答
  - 裂变：成就卡 / 家长周报 / 作品分享持续带量
- **关键指标**：DAU/MAU、社区互动率（评论/点赞）、分享拉新数、7 日/30 日留存
- **过关标准**：社区每天有新内容、互动不靠自己推；分享带来的新用户占比稳定上升

### 阶段 3 · 横向扩 10→100：从"古诗 App"到"启蒙园"
- **目标**：多科启蒙，把"园"填满，提高人均时长与黏性。
- **内容横向扩**（每科复用同一套：听/读/练/游戏/社区）：
  - 国学：三字经、弟子规、成语、论语、唐诗宋词
  - 识字、拼音
  - 英语启蒙儿歌
  - 数学思维启蒙
  - 绘本/听故事（磨耳朵已起步）
- **社区随之扩**：不只晒古诗，晒各种成长、各科作品。
- **关键指标**：人均使用科目数、多科使用占比、内容消费时长、跨科留存
- **注意**：**先专后宽**——古诗这一科跑透了再扩，别一上来摊大。

### 阶段 4 · 变现
- **会员**：精品课 / 无限内容 / 去广告 / 高级功能
- **单科 / 精品内容**付费
- **亲子活动、教具、周边**
- **关键指标**：付费率、ARPU、续费率、LTV
- **前提**：留存和社区先立住，再谈变现（没留存的变现是无源之水）

## 四、北极星指标

建议用 **「周活跃家长数」**（或家长周留存）。
理由：家长是真实使用 + 付费 + 传播的主体，这个指标同时反映"有没有用、留不留得住、传不传得开"。

## 五、护城河

1. **内容厚度**：多科 + 优质（音频/配图/故事/游戏），越攒越值钱。
2. **社区氛围**：家长之间的关系网和内容沉淀，别人复制不了。

## 六、风险与提醒

- **儿童内容合规**：UGC + 儿童照片敏感，必须**先审核后公开**（已实现），人工把关到位。
- **社区冷启动难**：0→1 最难，需要运营投入，不能指望自然增长。
- **内容生产成本**：横向扩科目=持续的内容/音频生产，要算成本。
- **别摊太宽**：宽是后劲，专是起手。先把一个点做透。

# 发布记录

## 2026.06.27.1 / 2026-06-27

### 新功能
- 🎧 **磨耳朵·古诗听单**：连续播放官方朗读，支持后台/锁屏播放、锁屏显示诗名封面、睡前定时（15/30/60 分钟自动停）。听单可切「全部 / 已学过 / 启蒙必背 / 进阶诗」。开辟睡前、路上等被动听诗场景。
- 📖 **今天读什么（每日学习计划）**：首页/悬浮球面板自动安排「1 首新诗 + 到期复习」，当天锁定、完成自动打勾。
- 🔁 **复习巩固**：学过的诗按间隔（≥2 天）提醒再读。
- 🎁 **邀请好友（裂变）**：分享带邀请码，好友首次进入即绑定邀请关系，邀请人解锁勋章（1/3/5/10 人四档）；被邀请者落地弹「好友 XX 邀请你」欢迎语。家长周报、成就卡、首页分享均带邀请码。
- 📊 **家长周报**：本周学习/复习汇总，可分享。
- 🔔 **睡前学习提醒**：每天傍晚订阅消息提醒，落地「磨耳朵」听单，引导睡前听一首。
- 📈 **数据看板（管理员）**：核心事件埋点 + 活跃用户 / 每日活跃 / 热门诗看板。
- 👥 首页社区人气条（真实在学人数）。

### 体验优化
- 发现页朗诵卡换用古诗配画封面 + 中央播放键，告别「一墙话筒」。
- 朗诵轮播改三列（诗 / 用户名 / 播放键）。
- 全面优化文案，去掉「功课 / 复习 / 打卡」等上学压力感，改为「读什么 / 再读 / 小书房」等轻松童趣措辞。
- 精简首页：删去重复的「继续学习」「今日小任务」，「今天怎么学」改为学习方式入口；悬浮球面板收敛为「连续 → 今日 → 磨耳朵 → 邀请」。
- 离开首页即停止发现页朗诵播放，杜绝后台串音。

### 修复
- 修复悬浮球面板邀请卡宽度被压缩、文字截断的问题。
